## 📄 Abstract

Plant diseases significantly affect agricultural productivity, making early detection and proper treatment very important. This project proposes a smart plant monitoring and disease detection system that integrates both hardware and software approaches.

The system consists of two main parts: hardware implementation and software-based image processing. In the hardware section, sensors continuously monitor environmental parameters such as temperature, humidity, soil moisture, and volatile gas levels. These values are displayed and tracked in real time. A threshold value of 200 is set for the gas sensor. When the gas level exceeds this limit, it indicates possible plant stress or disease, as plants release specific gases under such conditions. Upon detection, the system automatically activates a pump to supply fertilizer or treatment.

In the software section, a web-based application is used for plant disease detection through image processing. Users can upload plant images, and the system analyzes them to identify possible diseases.

By combining sensor-based monitoring with image-based analysis, the system provides an efficient and automated solution for early disease detection and management, improving plant health and agricultural productivity.
